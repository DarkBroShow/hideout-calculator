# Hideout Calculator

## Назначение
Калькулятор крафта для убежища в Stalcraft:X. Позволяет:
- Искать предметы по названию
- Строить дерево крафта (рекурсивно до 6 уровней)
- Анализировать ресурсы для производства
- Редактировать состав материалов

## Основные сценарии
1. **Поиск предмета** → открывается дерево крафта
2. **Анализ рецепта** → видно все ингредиенты и их дерево
3. **Редактор материалов** → изменение количества ресурсов

## Архитектура
```
game-data/stalcraft-database/
↓ (импорт через scripts)
PostgreSQL (items, recipes, recipe_ingredients)
↓ Express API (поиск + дерево рецептов)
Vue 3 + VueFlow (граф крафта + редактор)
```
